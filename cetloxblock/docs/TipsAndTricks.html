Xblock development undocumented stuff

  1. To update an installed xblock – upgrade the version number in setup.py and reinstall.
  2. Xblock.author_view is used by studio only if xblock.has_author_view = True .
  3. The editor's save/cancel buttons are part of the xblock's html! Including their binding to click and to studio_submit handler. (a good examle: simplevideo http://opencraft.com/doc/edx/xblock/tutorial.html)
  4. If missing, studio displays a 'close' button that only closes the editor.
  5. See simplevideo tutrial for runtime.notify('save'). It triggers 'saving…' label and a reloads author view when done.
  6. The xblock title in the studio is set by the field 'display_name' (scope.settings).
